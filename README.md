*  Personal Finance Insight Tracker  *

Live Demo  : ( https://varsha596.github.io/Personal-Finance-Insight-Tracker/ )

Description  :
A frontend web application to track income, expenses, and gain insights into personal finances. Users can view category-wise summaries and monitor monthly balance.

Features:  
Add and delete income/expense entries  
View monthly financial summary  
Track expenses by category  
Responsive and interactive UI  

Tech Stack:  
HTML, CSS, JavaScript  
LocalStorage for data persistence  

How to Use Locally:  
 Clone the repository
